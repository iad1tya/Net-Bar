Net Bar is more than just a network speed monitor - it's a lightweight, beautifully designed macOS menu bar app to track your network, battery, memory, storage, and processor performance in one place.

Download DMG Installer View on GitHub

Powerful Features

Monitor your Mac's health and network performance with real-time data and comprehensive diagnostics

Real-time Monitoring

View current download and upload speeds directly in the menu bar with live updates.

  • Live network speed tracking
  • Instant performance updates
  • Minimal resource usage

Comprehensive System Stats

Monitor all critical aspects of your Mac's performance in one unified dashboard.

  • Memory Usage: Track actual RAM usage and pressure
  • Processor Usage: Monitor CPU performance
  • Storage: Disk space (Used vs Free)
  • Battery: Real-time health and status
  • Temperature: Device thermal state

Native macOS Experience

Built with SwiftUI and AppKit for seamless integration with macOS.

  • Lightweight and crash-free
  • Optimized for macOS Sonoma+
  • Menu bar native integration
